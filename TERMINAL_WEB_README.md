# SberPOS Terminal (Web Version)

Модифицированное терминальное приложение с QR-кодами, указывающими на веб-сайт.

## Файл

`sberpos_terminal_web.apk` (20.8 MB)

## Изменения

- QR-коды теперь указывают на `https://sberpos-web.onrender.com/pay/{terminal_id}/key={key}`
- Вместо API сервера используется отдельный веб-сайт для страниц оплаты

## Установка

1. Удалите старую версию приложения (если установлена)
2. Установите `sberpos_terminal_web.apk`
3. Разрешите установку из неизвестных источников

## Использование

1. Запустите приложение
2. Войдите с учетными данными терминала
3. При оплате генерируется QR-код с уникальным ключом
4. QR-код ведет на веб-сайт с красивой страницей оплаты

## Технические детали

- Изменен URL в `MainActivity.smali`:
  - Было: `https://sberpos-api.onrender.com/pay/`
  - Стало: `https://sberpos-web.onrender.com/pay/`
- QR-код генерируется локально в приложении
- Формат URL: `/pay/{terminal_id}/key={6-digit-key}`

## Безопасность

- Каждый QR-код содержит уникальный 6-значный ключ
- Ключ проверяется на веб-сайте перед показом страницы оплаты
- Ключ инвалидируется после использования
