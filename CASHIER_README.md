# 💼 СберЭкран - Программа-касса

Графическое приложение для управления терминалами SberPOS с компьютера.

## 🚀 Установка

### Windows
```bash
pip install -r cashier_requirements.txt
python cashier_app.py
```

### Linux/Mac
```bash
pip3 install -r cashier_requirements.txt
python3 cashier_app.py
```

## 📋 Возможности

### Авторизация
- Вход по ID терминала и паролю
- Автоматическое сохранение сессии

### Быстрые суммы
- Кнопки для быстрой отправки: 100₽, 200₽, 500₽, 1000₽, 2000₽, 5000₽
- Поле для ввода своей суммы

### Управление оплатами
- ✅ Подтвердить карту
- ❌ Отклонить карту
- 🚫 Отменить операцию
- 📊 Статус терминала

### Журнал операций
- Все действия логируются с временными метками
- Возможность очистки лога

## 🔧 Настройки

Для изменения адреса сервера отредактируйте строку в `cashier_app.py`:
```python
self.base_url = "https://sberpos-api.onrender.com"
```

## 📝 Примечания

- Требуется Python 3.7+
- Работает на Windows, Linux, macOS
- Использует tkinter (встроен в Python)
- Все запросы выполняются в фоновых потоках (не блокируют интерфейс)

## 🐛 Решение проблем

### Ошибка "No module named 'tkinter'"
**Linux:**
```bash
sudo apt-get install python3-tk
```

**macOS:**
```bash
brew install python-tk
```

### Ошибка подключения
- Проверьте интернет-соединение
- Убедитесь что сервер доступен: https://sberpos-api.onrender.com
- Проверьте правильность ID терминала и пароля

## 📦 Создание исполняемого файла

### Windows (.exe)
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "СберЭкран-Касса" cashier_app.py
```

Готовый файл будет в папке `dist/`

### Linux/Mac
```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "sberpos-cashier" cashier_app.py
```

## 🎯 Горячие клавиши

- `Enter` в поле пароля - вход
- `Enter` в поле своей суммы - отправить

## 📞 Поддержка

При возникновении проблем проверьте журнал операций в правой панели приложения.
